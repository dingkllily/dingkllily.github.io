I-BALLAST Mobile is an iOS application for field ballast inspection designed to support image processing and model inference in resource-constrained environments.

- Built with SwiftUI and Core ML for production-oriented field workflows.
- Supports real-time image processing and on-device inference.
- Integrated with a remote-service workflow for higher-quality DINO-SAM inference when needed.
